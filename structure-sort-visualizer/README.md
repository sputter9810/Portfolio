# Structure & Sorting Visualizer

A frontend visualizer for common sorting algorithms and core data structures.  
Built as a portfolio project to demonstrate algorithmic thinking, clean UI design, and TypeScript-based implementation.

## Features

### Sorting Visualizer

Visualizes common sorting algorithms using animated bar charts.

Planned sorting algorithms:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

Stretch algorithms:

- Shell Sort
- Counting Sort
- Radix Sort

### Data Structure Visualizer

Demonstrates common data structures and basic operations.

Planned data structures:

- Array
- Linked List
- Stack
- Queue
- Hash Table
- Binary Search Tree

Supported operations may include:

- Add
- Find
- Delete
- Traverse
- Breadth-first search
- Depth-first search

## Tech Stack

- React
- TypeScript
- Vite
- CSS

## Project Goals

The goal of this project is to provide a clear visual demonstration of common computer science concepts including sorting algorithms and data structures.

This project is designed to be:

- Quick to build
- Easy to extend
- Cleanly structured
- Suitable for a software engineering portfolio

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development Server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the production build:
```
npm run preview
```
